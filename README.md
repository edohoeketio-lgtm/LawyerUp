# LawyerUp

A legal services platform connecting users with verified legal professionals. Built with Next.js and TypeScript for fast, accessible legal consultations.

## Features

- Lawyer discovery and search
- Profile pages with specializations and availability
- Consultation booking flow
- Responsive design

## Tech Stack

- Next.js (App Router)
- TypeScript
- CSS Modules

## Running Locally

```
npm install
npm run dev
```
